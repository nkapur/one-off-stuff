# one-off-stuff
One off coding adventures when learning, hacking or leetcoding

Create you granular virtual environments with the right prompt names. For example,

```
% python -m venv .venv --prompt 'ai-agent-venv'
```

On the commandline, this would show which venv is activated

```
navneetkapur@mac ai-agent % va
(ai-agent-venv) navneetkapur@mac ai-agent %
```
